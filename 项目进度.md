## 技术架构

```
前端：Next.js 16 + TypeScript + Tailwind CSS v4
数据库：Supabase (PostgreSQL)
认证：Clerk
AI：OpenAI API
部署：Cloudflare Pages
```

## 项目结构

```
prompt-genius/
├── src/
│   ├── app/
│   │   ├── page.tsx          # 首页
│   │   ├── generate/page.tsx # 提示词生成页
│   │   ├── templates/page.tsx # 模板库
│   │   ├── optimize/page.tsx  # 提示词优化器
│   │   ├── pricing/page.tsx   # 定价页
│   │   ├── history/page.tsx   # 历史记录
│   │   └── api/
│   │       ├── generate/route.ts
│   │       └── optimize/route.ts
│   ├── components/
│   │   ├── Header.tsx         # 服务端 Header
│   │   ├── ClientHeader.tsx   # 客户端 Header
│   │   ├── HeaderAuth.tsx     # Clerk 认证组件
│   │   ├── Footer.tsx
│   │   ├── CategoryCard.tsx
│   │   ├── TemplateCard.tsx
│   │   └── PromptGenerator.tsx
│   ├── data/
│   │   └── templates.ts       # 模板数据
│   ├── lib/
│   │   ├── supabase.ts        # Supabase 客户端
│   │   └── utils.ts           # 工具函数
│   ├── services/
│   │   └── supabase-service.ts # 业务逻辑层
│   ├── types/
│   │   └── index.ts           # TypeScript 类型
│   └── middleware.ts           # Clerk 中间件
├── sql/
│   └── init.sql               # 数据库建表语句
├── wrangler.jsonc             # Cloudflare 部署配置
└── .env.local                 # 环境变量
```
